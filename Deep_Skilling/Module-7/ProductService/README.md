# ProductService

This service is the Web API from Module 6 (`ProductsController`, `AuthController`, EF Core
setup from Module 5), running independently on its own port (e.g. https://localhost:5001).

Copy the contents of `Module-6/` (and the `Models`/`Data`/`Repositories` folders from
`Module-5/`) into this folder to stand it up as its own deployable service.
