# Module 14 - Gen AI Fundamentals

## What this demonstrates
- `genai-lab/prompt-engineering-exercise.md` - four hands-on prompts
  covering zero-shot, few-shot, and chain-of-thought prompting, plus a
  "fix a vague prompt" exercise, each with concrete things to observe and
  compare in the model's output
- `genai-lab/copilot-hands-on.md` - four GitHub Copilot exercises that
  deliberately reuse code from earlier modules: generating a function from
  a comment, generating documentation for an existing method, reviewing
  the intentionally buggy Module 9 component for bugs, and generating
  NUnit test cases - each with a "check/compare" prompt to make sure you
  review the AI's output rather than accepting it blindly

## To run
1. Read the technique descriptions in `prompt-engineering-exercise.md` and
   run each prompt yourself in GitHub Copilot Chat (or any available
   chat-based coding assistant), noting the differences described.
2. Install/sign in to GitHub Copilot in VS Code, then work through
   `copilot-hands-on.md` end to end - it deliberately points back at
   Module 4 (NUnit), Module 5 (EF Core repository), and Module 9 (the
   buggy component) so you can compare Copilot's output against code you
   already understand.
3. Complete the reflection prompt at the end of `copilot-hands-on.md` -
   this is the most important part of the exercise for the module's
   learning objective on responsible AI-assisted coding.
